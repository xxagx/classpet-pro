# ClassPet Pro 使用说明

## 系统简介
班级互动宠物养成系统，将学生积分转化为可视化宠物进化游戏，提升课堂趣味性。

## 快速开始

### 1. 启动系统
```bash
# 方式1：直接双击打开（仅本地使用）
双击 web/index.html

# 方式2：启动服务器（推荐，支持多设备访问）
cd web
python3 -m http.server 8080

# 然后浏览器访问
大屏：http://localhost:8080
手机：http://[电脑IP]:8080/admin.html
```

### 2. 大屏端使用（学生互动）
1. 打开 `index.html`
2. 全屏显示（按F11）
3. 学生点击自己的宠物卡片
4. 老师点击加减分按钮
5. 达到进化条件自动播放动画

### 3. 手机管理端（老师操作）
1. 手机浏览器打开 `admin.html`
2. 查看学生列表和积分
3. 点击"批量加分"快速给多个学生加分
4. 点击学生可编辑详情
5. 点击"班级设置"修改进化阈值和奖励

### 4. 教师模式（大屏设置）
1. 大屏点击"教师模式"
2. 输入密码：1234（默认）
3. 可修改班级设置

## 数据管理

### 导出数据
手机端点击"导出数据"：
- Excel格式：用于表格分析
- JSON格式：完整数据备份

### 数据存储
所有数据保存在浏览器本地，清除浏览器数据会丢失。
建议定期导出JSON备份。

## 自定义配置

### 修改进化阈值
教师模式 → 进化阶段 → 修改数值 → 保存

### 添加奖励
教师模式 → 奖励设置 → 添加奖励 → 保存

### 修改密码
教师模式 → 基础设置 → 修改密码 → 保存

## 常见问题

**Q: 手机访问不了？**
A: 确保手机和电脑在同一WiFi下，使用电脑IP地址访问。

**Q: 数据丢失了？**
A: 浏览器清除数据会导致丢失，建议定期导出备份。

**Q: 如何添加更多学生？**
A: 手机管理端 → 学生管理 → 点击右上角"+添加"

**Q: 宠物图片能换吗？**
A: 当前版本使用emoji，后续版本支持自定义图片。
