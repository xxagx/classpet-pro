# Seedance 画面 Prompts 模板

## 风格基调
- 科技感、冷色调、都市夜景
- 避免人物特写，多用场景和抽象画面
- 单镜头 3-5 秒

## 常用镜头类型

### 1. 城市夜景
```
Aerial view of modern city at night, neon lights reflecting on wet streets, cyberpunk atmosphere, cool blue and purple tones, cinematic wide shot, 4K
```

### 2. 办公室/会议室
```
Empty modern conference room at night, floor-to-ceiling windows showing city skyline, minimalist interior, soft ambient lighting, professional atmosphere
```

### 3. 数据流动画
```
Abstract data visualization, flowing lines of light representing information networks, dark background with glowing particles, technological aesthetic
```

### 4. 科技风镜头
```
Futuristic interface hologram floating in dark space, transparent screens with charts and graphs, blue and white glow, sci-fi aesthetic
```

### 5. 抽象经济画面
```
Abstract representation of financial markets, ascending and descending geometric shapes, gold and blue color scheme, minimalist composition
```

### 6. 思考/决策氛围
```
Silhouette of person looking at cityscape through window, contemplative mood, dramatic backlighting, urban environment
```

## 使用原则
- 每段视频用 5-8 个不同镜头
- 根据脚本内容匹配画面情绪
- 保持整体视觉一致性（冷色调为主）
