## 🎨 视觉风格：「咨询顾问的笔记本」

### 设计理念
不是"精致的企业宣传片"，而是"专家在给你讲干货"

### 具体呈现
```
背景：仿纸张纹理 #f5f5f0（米白）或深色黑板 #2c3e50
元素：手绘箭头、荧光笔标记、便利贴便签
字体：手写体标题 + 黑体正文（混搭更真实）
动画：几乎为零，或简单的淡入淡出
```

### 参考效果
- 像顾问在白板上随手画给你的分析图
- 像你看过的麦肯锡内部汇报PPT
- 像得到App课程的知识卡片

## 🎙️ 配音方案

### 推荐：剪映「文字转语音」
**路径：** 剪映 → 文本 → 朗读 → 选择「解说小帅」或「新闻男声」

**优点：**
- 完全免费
- 无需信用卡
- 语速可调（建议1.2倍速，更有信息量）

### 进阶：讯飞配音（如需更高质感）
- 网址：peiyin.xunfei.cn
- 价格：约0.5元/100字（一条视频约2-3元）
- 声音：「知意」或「知远」（商务感强）

## ✂️ 剪映剪辑要点

### 时间分配
```
总时长：85秒
片头hook：3秒（必须抓眼球）
问题放大：10秒
数据论证：30秒（核心信息密度）
方法论：20秒
案例：10秒
CTA：10秒
片尾：2秒
```

### 节奏控制
- **每5-8秒必须有变化**（换页、数字跳动、箭头出现）
- **关键数字停顿1秒**，让观众消化
- **全程无复杂转场**，直接切或淡入淡出

### BGM选择
- 类型：轻电子、商务感
- 音量：15-20%（能听见但不干扰）
- 推荐：剪映音乐库搜「轻快」「商务」「科技」

## 💡 去AI化的关键细节

1. **故意不完美**：手写字稍微歪一点，更像真人
2. **加入"我"的视角**："我见过一个酒店..."而不是"某酒店..."
3. **具体数字**：不说"很多酒店"，说"杭州某120间客房酒店"
4. **口语化表达**："说白了...""你可能想不到..."
5. **适度留白**：不要塞满画面，给观众思考空间
