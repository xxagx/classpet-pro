🔍 GitHub Pages 配置步骤

  1. 访问 GitHub 仓库
  2. 点击 Settings
  3. 找到 Pages 选项
  4. 配置：Deploy from a branch
  5. Branch: main
  6. Folder: / (root)
  7. 点击 Save
  8. 等待 1-5 分钟部署完成
