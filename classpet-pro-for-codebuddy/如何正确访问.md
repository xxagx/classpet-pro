# 🚨 重要：如何正确访问 ClassPet Pro

## ❌ 错误的做法

**不要直接双击打开这个文件！**
```
file:///C:/Users/贝贝/Desktop/classpet-pro1.0-main%20(1)/classpet-pro1.0-main/classpet-pro-for-codebuddy/mobile.html
```

这样会导致：
- ❌ 按钮点击无响应
- ❌ 功能无法使用
- ❌ 数据无法保存

---

## ✅ 正确的做法（3 种方法）

### 方法 1: 使用"快速启动.bat"（最简单）⭐

1. 双击这个文件：
   ```
   快速启动.bat
   ```

2. 浏览器会自动打开到正确地址

3. 点击"开启旅程"按钮即可使用

**优点**: 一键启动，自动检查服务器

---

### 方法 2: 手动访问（推荐）

1. 确保服务器在运行（查看任务栏是否有黑色窗口）

2. 在浏览器地址栏输入：
   ```
   http://127.0.0.1:8080/mobile.html
   ```

3. 按回车键打开

4. 点击"开启旅程"按钮

**注意**: 地址必须是 `http://` 开头，不是 `file://`！

---

### 方法 3: 使用 start.html（方便）

1. 双击打开：
   ```
   start.html
   ```

2. 会自动跳转到正确地址

**原理**: start.html 会自动重定向到 HTTP 服务器

---

## 🔍 如何判断是否正确使用？

### ✅ 正确的地址（可以正常使用）
```
http://127.0.0.1:8080/mobile.html
```
- 以 `http://` 开头
- 包含 IP 地址 `127.0.0.1` 或 `localhost`
- 包含端口号 `:8080`

### ❌ 错误的地址（无法使用）
```
file:///C:/Users/.../mobile.html
```
- 以 `file://` 开头
- 包含本地路径 `C:/Users/`
- 没有端口号

---

## 📱 手机访问方法

### 步骤 1: 确保在同一 WiFi

- 电脑和手机连接到同一个 WiFi 网络

### 步骤 2: 查看电脑 IP 地址

1. 按 `Win + R`
2. 输入 `cmd` 并回车
3. 输入 `ipconfig` 并回车
4. 找到 `IPv4 地址`（如 `192.168.1.237`）

### 步骤 3: 在手机浏览器访问

```
http://电脑的 IPv4 地址:8080/mobile.html
```

例如：
```
http://192.168.1.237:8080/mobile.html
```

---

## 🎯 快速验证

### 测试按钮是否正常

1. 访问正确地址后
2. 点击"开启旅程 🎵"按钮
3. 应该看到：
   - ✅ 欢迎弹窗关闭
   - ✅ 登录界面显示
   - ✅ 可以选择学生姓名

如果还是无响应，请检查：
1. 地址是否以 `http://` 开头
2. 是否包含 `:8080` 端口
3. 浏览器控制台是否有错误（按 F12）

## 💡 常见问题

### Q1: 服务器未运行怎么办？

**A**: 双击"启动服务器.bat"或"快速启动.bat"

### Q2: 端口 8080 被占用？

**A**: 
1. 关闭其他占用 8080 端口的程序
2. 或修改为其他端口（如 8081）

### Q3: 手机无法访问？

**A**: 
1. 检查防火墙设置
2. 确保在同一 WiFi
3. 确认 IP 地址正确

### Q4: 如何停止服务器？

**A**: 
- 找到黑色的命令行窗口
- 按 `Ctrl + C`
- 或关闭窗口
