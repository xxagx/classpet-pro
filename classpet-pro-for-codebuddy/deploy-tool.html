⚡ Vercel 部署步骤（5 分钟）

1

上传代码到 GitHub

首先需要将代码上传到 GitHub 仓库

2

部署到 Vercel

使用 GitHub 账号登录 Vercel 并导入项目

提示：登录后点击 "Add New Project" → 选择你的仓库 → 点击 "Deploy"
3

获取访问链接

部署完成后，您会获得一个免费域名

⚡ Netlify 部署步骤（3 分钟）

1

打开 Netlify

访问 Netlify 官网并注册账号

2

拖拽上传

将整个项目文件夹拖入 Netlify 上传区域

文件夹位置：
c:\Users\贝贝\Desktop\classpet-pro1.0-main (1)\classpet-pro1.0-main\classpet-pro-for-codebuddy
3

获取访问链接

上传完成后获得免费域名

⚡ GitHub Pages 部署步骤（5 分钟）

1

创建 GitHub 仓库

创建名为 `你的用户名.github.io` 的仓库

2

上传代码

将所有文件上传到仓库

git init
git add .
git commit -m "Initial commit"
git push -u origin main
3

启用 Pages

Settings → Pages → 选择 main branch → Save

📱 生成访问二维码

扫描二维码即可访问系统

⚠️ 重要提示

数据存储说明：

• 当前版本使用 localStorage 存储数据

• 数据保存在用户浏览器本地

• 清除浏览器缓存会丢失数据

• 建议定期在教师端导出数据备份

访问优化：

• Vercel/Netlify 是境外服务器，国内访问可能稍慢

• 建议生成二维码让学生扫码访问

• 可以绑定自定义域名提升访问速度
