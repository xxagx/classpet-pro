# 🎮 ClassPet Pro 8 阶段进化系统 - 完整说明文档

## 📋 系统概述

ClassPet Pro 8 阶段进化系统是一个完整的宠物成长体系，模拟了宠物从孵化到成为神话生物的完整生命周期。每个阶段都有独特的视觉特征、能力解锁和成长指标。

## 🎨 视觉特效系统

### 边框动画效果

每个阶段都有独特的 CSS 动画效果：

1. **simple-glow**: 柔和的绿色光晕，象征生命萌芽
2. **star-twinkle**: 蓝色星光闪烁，象征智慧启蒙
3. **bounce-light**: 黄色跳动光效，象征活力成长
4. **rotate-halo**: 橙色旋转光环，象征能量觉醒
5. **particle-burst**: 紫色粒子爆发，象征力量释放
6. **aura-wave**: 金色气场波纹，象征大师威严
7. **meteor-trail**: 粉红流星拖尾，象征传奇降临
8. **divine-glory**: 多彩神圣光辉，象征神话降临

### 进化过渡特效

当宠物从一个阶段进化到下一个阶段时，会触发全屏进化特效：
- 进化光芒（expand-and-fade）
- 粒子效果（floating particles）
- 阶段转换文字动画
- 持续 3 秒的沉浸式体验

## 📊 进度系统

### 进度计算

```javascript
进度百分比 = (当前积分 - 阶段最小积分) / (阶段最大积分 - 阶段最小积分) × 100%
```

### 升级提示

- 显示当前积分和阶段上限
- 显示距离下一阶段所需积分
- 已达最高阶段时显示特殊标识

## 💻 技术实现

### 核心类

#### 1. EvolutionManager (进化管理器)
```javascript
// 获取当前阶段
getStageByScore(score)

// 获取下一阶段
getNextStage(currentKey)

// 计算进度百分比
getProgressPercentage(score, currentStage)

// 获取升级所需分数
getScoreForNextStage(score, currentStage)

// 渲染阶段显示
renderStageDisplay(container, score, petEmoji)

// 创建进化特效
createEvolutionEffect(container, fromStage, toStage)
```

#### 2. PetImageManager (宠物图片管理器)
```javascript
// 获取当前风格的宠物数据
getCurrentPets()

// 获取所有风格
getAllStyles()

// 上传自定义图片
uploadImage(stageIndex, file)

// 获取宠物显示内容
getPetDisplay(stageName, size)
```

### CSS 动画系统

所有视觉特效都通过 CSS 动画实现：
- 使用 `@keyframes` 定义动画序列
- 使用 `animation` 属性应用动画
- 支持无限循环和单次触发
- 响应式设计，适配不同屏幕

## 🎯 使用示例

### 示例 1: 查看学生宠物阶段
```javascript
const student = { name: '张三', score: 350 };
const evolutionManager = new EvolutionManager();
const stage = evolutionManager.getStageByScore(student.score);
console.log(`${student.name} 的宠物处于：${stage.name}`);
// 输出：张三 的宠物处于：进化学
```

### 示例 2: 计算升级进度
```javascript
const score = 600;
const currentStage = evolutionManager.getStageByScore(score);
const progress = evolutionManager.getProgressPercentage(score, currentStage);
const needed = evolutionManager.getScoreForNextStage(score, currentStage);

console.log(`进度：${progress.toFixed(1)}%`);
console.log(`还需：${needed} 分`);
// 输出：进度：40%
//       还需：150 分
```

### 示例 3: 触发进化特效
```javascript
const fromStage = evolutionManager.getStageByScore(500);
const toStage = evolutionManager.getStageByScore(501);
const container = document.body;

evolutionManager.createEvolutionEffect(container, fromStage, toStage);
```

## 🎨 自定义扩展

### 添加新阶段
在 `EVOLUTION_STAGES` 配置对象中添加新阶段：
```javascript
const EVOLUTION_STAGES = {
    // ... 现有阶段
    stage9: { 
        name: '超神期', 
        minScore: 2000,
        maxScore: Infinity,
        size: 130,
        frameColor: '#00FFFF',
        frameEffect: 'cosmic-aura',
        description: '超越神话的存在'
    }
};
```

### 添加新特效
在 CSS 文件中定义新的 `@keyframes` 动画：
```css
@keyframes cosmic-aura {
    0%, 100% {
        box-shadow: 0 0 50px #00FFFF, 0 0 100px #00FFFF;
    }
    50% {
        box-shadow: 0 0 100px #00FFFF, 0 0 200px #00FFFF;
    }
}
```
