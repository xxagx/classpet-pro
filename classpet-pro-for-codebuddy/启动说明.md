# 🚀 ClassPet Pro 启动说明

**重要**: 请勿直接双击打开 HTML 文件！必须通过 HTTP 服务器访问。

---

## ⚠️ 为什么不能直接打开 HTML 文件？

直接双击打开（`file://` 协议）会导致：
- ❌ JavaScript 无法加载
- ❌ CSS 样式丢失
- ❌ 按钮点击无响应
- ❌ 数据无法保存
- ❌ 音频无法播放

**原因**: 现代浏览器出于安全考虑，限制了本地文件访问。

---

## 📱 手机访问方法

### 步骤 1: 确保在同一 WiFi 网络

- 电脑和手机连接到同一个 WiFi
- 记下电脑的 IP 地址（如 `192.168.1.237`）

### 步骤 2: 在手机浏览器访问

```
http://电脑的 IP:8080/mobile.html
```

例如：
```
http://192.168.1.237:8080/mobile.html
```

### 步骤 3: 添加到主屏幕（可选）

- Safari: 分享 → 添加到主屏幕
- Chrome: 菜单 → 添加到主屏幕

## 🔧 常见问题

### Q1: 服务器启动失败？

**A**: 检查端口 8080 是否被占用
```bash
# 使用其他端口
npx http-server -p 8081
```

### Q2: 手机无法访问？

**A**: 
1. 检查防火墙设置
2. 确保在同一 WiFi 网络
3. 检查 IP 地址是否正确

### Q3: 点击按钮无响应？

**A**: 
1. 确保通过 HTTP 访问（不是 file://）
2. 清除浏览器缓存
3. 检查浏览器控制台（F12）

### Q4: 如何停止服务器？

**A**: 
- 在命令行窗口按 `Ctrl+C`
- 或关闭命令行窗口
