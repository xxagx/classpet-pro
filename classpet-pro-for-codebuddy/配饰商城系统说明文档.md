# 🛍️ ClassPet Pro 积分兑换配饰系统 - 完整说明文档

## 📋 系统概述

ClassPet Pro 积分兑换配饰系统是一个完整的虚拟商品交易和装扮系统，允许学生用户使用积分从商城购买各类配饰商品，并为自己的宠物购买、穿戴和替换配饰。

## 🔄 积分兑换流程

### 步骤 1: 浏览商品
- 学生用户进入商城界面
- 可按分类筛选（全部、衣服、鞋子、帽子等）
- 可按稀有度筛选（全部、普通、稀有、史诗、传说）
- 查看商品详情：名称、分类、稀有度、描述、所需积分

### 步骤 2: 积分验证
- 系统自动检测用户当前积分
- 积分充足：显示"购买"按钮（绿色）
- 积分不足：显示"积分不足"按钮（灰色，禁用）

### 步骤 3: 购买确认
点击购买按钮后，弹出确认对话框，显示：
- 商品图片（emoji）
- 商品名称和分类
- 商品价格
- 当前积分余额
- 购买后剩余积分

### 步骤 4: 完成购买
- 确认购买：扣除积分，添加商品到库存
- 取消购买：关闭对话框，不做任何操作
- 购买成功：显示成功提示，更新积分显示

## 🎒 配饰库存管理

### 库存界面功能

#### 当前穿戴区域
- 显示 8 个穿戴槽位（衣服、鞋子、帽子、眼镜、发卡、背包、首饰、手链）
- 已穿戴：显示配饰图片和名称，可脱下
- 未穿戴：显示"未穿戴"提示

#### 配饰列表
- 标签分类：全部、衣服、鞋子、帽子等
- 显示所有已购买的配饰
- 已穿戴配饰显示特殊边框和"已穿戴"标识
- 显示购买时间（今天/昨天/具体日期）

### 穿戴功能

#### 穿戴配饰
1. 在库存中点击"穿戴"按钮
2. 系统自动应用到对应分类的穿戴槽位
3. 如果该槽位已有配饰，自动替换
4. 显示成功提示
5. 刷新库存显示

#### 脱下配饰
1. 在当前穿戴区域点击"脱下"按钮
2. 配饰从槽位移除
3. 显示成功提示
4. 刷新库存显示

#### 替换配饰
1. 在库存中点击未穿戴的配饰
2. 自动替换同分类的当前穿戴配饰
3. 显示成功提示
4. 更新显示

## 📊 系统集成

### 引入文件

```html
<!-- 引入 CSS -->
<link rel="stylesheet" href="assets/css/accessory-system.css">

<!-- 引入 JS -->
<script src="js/accessory-mall.js"></script>
<script src="js/user-accessory-manager.js"></script>
```

### 初始化示例

```javascript
// 创建商城管理器
const mallManager = new AccessoryMallManager();

// 创建用户配饰管理器
const userAccessorManager = new UserAccessoryManager(studentId);

// 渲染商城
mallManager.renderMall(container, userPoints, handlePurchase);

// 渲染库存
userAccessorManager.renderInventory(container, handleWear, handleRemove);
```

### 购买流程示例

```javascript
function handlePurchase(accessory) {
    // 1. 验证积分
    if (userAccessorManager.getPoints() < accessory.points) {
        alert('积分不足！');
        return;
    }
    
    // 2. 执行购买
    const result = userAccessorManager.purchaseAccessory(accessory);
    
    // 3. 处理结果
    if (result.success) {
        showSuccessToast(`成功购买 ${accessory.name}！`);
        // 4. 刷新界面
        renderMall();
        renderInventory();
    }
}
```

### 穿戴流程示例

```javascript
function handleWear(accessoryId) {
    // 1. 执行穿戴
    const result = userAccessorManager.wearAccessory(accessoryId);
    
    // 2. 处理结果
    if (result.success) {
        showSuccessToast(result.message);
        // 3. 刷新界面
        renderInventory();
        // 4. 更新宠物显示
        updatePetDisplay();
    }
}
```

## 🔧 故障排查

### 问题 1: 积分不更新
**解决方案：** 检查 LocalStorage 是否正确保存，清除缓存重试

### 问题 2: 购买后配饰未添加到库存
**解决方案：** 检查 purchaseAccessory 方法返回值，查看控制台错误

### 问题 3: 穿戴状态未保存
**解决方案：** 确认 saveUserData 方法被调用，检查浏览器存储权限

### 问题 4: 界面不刷新
**解决方案：** 检查 render 方法是否正确调用，DOM 元素是否存在
