# 2026-03-13 - Kling + CapCut 视频制作流程测试

## 视频生成方案讨论

### 工具选择
- **Kling AI（可灵）**：AI视频生成
- **CapCut（剪映）**：视频剪辑

### 制作流程（已验证）

#### 第一步：Kling 文生视频
- 打开 https://klingai.com/
- 选择「文生视频」
- 输入 Prompt（5秒，9:16竖屏）
- 等待生成（免费版排队约2小时）

**测试用 Prompt（方案B）：**
```
成都糖酒会展会现场，参展商数量锐减，空旷的展厅与往年热闹形成对比。镜头推移至三家酒店：豪华大堂、设计师民宿、社区精品酒店，客人络绎不绝。光影对比强烈，展现行业分化。4K画质，纪录片风格。
```

**生成结果：**
- 文件名：`kling_20260313_作品_成都糖酒会展会现场__2764_0`
- 效果：待评估

#### 第二步：CapCut 剪辑
1. 新建项目（9:16竖屏）
2. 导入 Kling 生成的视频
3. 添加字幕（标题 + 关键词）
4. 添加配音（CapCut AI 配音或 Edge-TTS）
5. 导出最终视频
