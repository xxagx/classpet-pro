## 新增模块设计（2026-02-26 晚）

### 模块一：商旅动态监测
**目标**：外部趋势校准信号，判断商旅需求方向与传导节奏  
**输入**：
- 本周商旅在住量（或区间）
- 环比趋势（↑ ↓ 持平）
- Weekly → To Day 变化

**判断逻辑**：
- 连续上升 → 商旅需求增强
- 连续下降 → 商旅需求走弱
- 商旅上升但 OTA 未涨 → 结构性升温
- 商旅下降但价格仍高 → 价格滞后风险

**输出**：商旅需求状态 + 是否存在价格传导滞后

### 模块二：竞品热度 + 市场转折预警
**目标**：通过抓取非样本竞品未来14天基础房型价格，判断供需压力与竞争强度  
**监测指标**：
1. 调价频率（是否频繁涨价）
2. 涨价后是否快速回调
3. D-3 是否出现明显关房
4. 未来14天价格曲线斜率变化

**判断逻辑**：
- 频繁涨价且不回调 → 需求真实增强
- 频繁涨价但快速回落 → 试探性涨价
- 连续下移 → 弱势阶段预警
- 提前量价格稳定抬升 → 升温信号

**输出**：
- 市场供需压力等级（平衡 / 紧张 / 高压）
- 是否存在转折信号

**原则**：不推算单店入住率、不输出单家对比、所有结果为趋势判断与决策辅助
